# InstantBooking
InstantBooking is a Hotel Reservation/Booking Application built using React, Node.js, Express, MongoDB, ejs, Mongoose Context API, JWT, custom hooks, and React calendar. We can give Search Inputs and Price Range and its will fetch the List containing full hotel information with images and also calculates the price according to date range and room numbers.It also has Authentication (Login / Register)
 It has Node MongoDB Authentication (Login / Register) and many more functionalities
It also has an Admin Dashboard and we will be able to delete items and create new ones
